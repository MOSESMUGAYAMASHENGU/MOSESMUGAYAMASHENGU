#### 📌 **Tech Matcher — Domain-Savvy Talent Matching (Python + SQL + Streamlit)**

**The Problem:**  
Modern hiring systems miss a powerful, growing talent group:  
**domain experts who are now integrating tech into their field**  
(e.g., development practitioners learning Python, public-health workers learning SQL, economists learning Power BI, etc.)

Traditional ATS filters hide them because they lack CS degrees — despite having rare hybrid value.

These “boundary spanners” are incredibly valuable but almost invisible.

**The Solution:**  
Tech Matcher is a lightweight, data-driven system that helps recruiters discover **domain-aware tech talent** by connecting:

- user profiles  
- technical skill progression  
- domain context relevance  

Using **Python (logic), SQL (storage), and Streamlit (UI)**, it surfaces upskillers & self-taught candidates who don’t appear in traditional tech searches but are perfect fits for NGOs, public-sector orgs, social enterprises, and tech-for-good teams.

**Why it matters:**  
Organizations increasingly need tech-capable people who understand real-world systems.  
Tech Matcher makes this talent visible.

*(MVP currently in development — focused on minimalism, clarity, and practical utility.)*
